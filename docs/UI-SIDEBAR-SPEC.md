## Home 按钮（左上角）

点击弹出下拉菜单：

```
┌─────────────────────┐
│ 客户A 160㎡别墅  ✓  │  ← 当前项目
│ 客户B 90㎡公寓      │
│ 样板间模板          │
├─────────────────────┤
│ + 新建项目          │
│ 📁 管理项目         │
└─────────────────────┘
```

- 切换项目 = 加载另一个 scene graph
- 新建项目 = 空白场景
- 管理项目 = 跳转项目列表页（未来做）

## 6 个功能入口

### 1. 设备

**左侧面板**：9 子系统筛选 → 设备产品列表 → 点击选中 → 在 3D 中放置

```
┌─────────────────┐
│ 💡灯光 📦面板 ...│  ← 子系统 chip 筛选
├─────────────────┤
│ 筒灯 DL-01      │
│   嵌入式 12W     │
│ 射灯 SL-01      │
│   轨道式 7W      │
│ 灯带 LS-01      │
│   暗槽 24V      │
│ ...              │
└─────────────────┘
```

**底部工具栏**：放置模式工具（吸附/旋转/高度调整）

**右侧面板**：选中设备的属性（亮度/色温/角度/范围等）

**3D 视图变化**：所有设备可见，选中子系统的设备高亮

---

### 2. 建筑

**左侧面板**：Pascal 原有的项目树（楼层→墙/门窗/楼板）

```
┌─────────────────┐
│ + 添加楼层        │
│ ▸ 1F             │
│ ▾ Level 0        │
│   + 上传底图      │
│   + 参考图        │
├─────────────────┤
│ [结构][家具][区域] │  ← Pascal 原有 Tab
├─────────────────┤
│ 墙体             │
│ 楼板             │
│ 天花             │
│ ...              │
└─────────────────┘
```

**底部工具栏**：Pascal 原有的结构/家具/区域工具

**右侧面板**：选中建筑元素的属性

**3D 视图变化**：标准建筑编辑视图

---

### 3. 拓扑

**左侧面板**：设备连接关系（按协议/系统分组）

```
┌─────────────────┐
│ KNX 总线         │
│   网关 → 执行器1 │
│        → 执行器2 │
│        → 面板1   │
│ 以太网           │
│   路由器 → AP1   │
│          → AP2   │
│ Zigbee           │
│   网关 → 传感器1 │
│        → 灯具1   │
│ 强电回路          │
│   配电箱 → 回路1 │
│          → 回路2 │
│ HVAC 管路         │
│   主机 → 出风口1 │
└─────────────────┘
```

**底部工具栏**：连线工具（拖拽设备之间建立连接）

**右侧面板**：选中连接的属性（协议/线材/功率）

**3D 视图变化**：设备之间显示连线，按协议不同颜色

---

### 4. 场景

**左侧面板**：场景列表 + 选中场景的设备配置

```
┌─────────────────┐
│ + 新建场景        │
├─────────────────┤
│ 🏠 回家模式      │
│ 🌙 睡眠模式      │
│ 🎬 影院模式      │
│ 🚪 离家模式      │
├─────────────────┤
│ 回家模式 — 设备   │
│ ✓ 客厅筒灯 80%   │
│ ✓ 灯带 40% 暖白  │
│ ✓ 窗帘 关闭      │

```

**底部工具栏**：播放控制（▶ 播放 / ⏸ 暂停 / ⏹ 停止 / 时间轴）

**右侧面板**：时间线编辑器（设备状态随时间变化的编排）

**3D 视图变化**：播放时设备按时间线联动

## 技术实现

### 改动范围

| 文件 | 改什么 |
|------|--------|
| `icon-rail.tsx` | 6 个图标入口替代原有 3 个 |
| `app-sidebar.tsx` | 新增 DevicePanel / TopologyPanel / ScenePanel / SharePanel / ReviewPanel |
| `tab-bar.tsx` | 建筑模式下仍用 [结构][家具][区域]，其他模式不显示 |
| `editor/index.tsx` | renderTabContent 适配 6 种面板 |

### 新增组件

```
packages/editor/src/components/ui/sidebar/panels/
  ├── site-panel/          ← 改名为 building-panel（建筑）
  ├── device-panel/        ← 新增（设备）
  ├── topology-panel/      ← 新增（拓扑）
  ├── scene-panel/         ← 已有骨架（场景）
  ├── share-panel/         ← 新增（分享）
  ├── review-panel/        ← 新增（查看）
  └── settings-panel/      ← 保留（设置）
```

### Home 下拉

新增 `packages/editor/src/components/ui/sidebar/project-switcher.tsx`

## 开发优先级

| 优先级 | 功能 | Sprint |
|--------|------|--------|
| P0 | 设备（目录+放置） | S1 |
| P0 | 建筑（Pascal 原有，改名） | S1 |
| P1 | 场景（编排+播放） | S3 |
| P2 | 查看（清单+报价） | S4 |
| P2 | 分享（链接+权限） | S5 |
| P3 | 拓扑（连线+协议） | S4+ |
